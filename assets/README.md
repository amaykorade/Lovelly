# Assets Directory

This directory contains all static assets for your React Native app.

## Required Assets

Based on your `app.json` configuration, you'll need:

1. **icon.png** (1024x1024px) - App icon
2. **splash.png** (1284x2778px recommended) - Splash screen image
3. **adaptive-icon.png** (1024x1024px) - Android adaptive icon
4. **favicon.png** (48x48px) - Web favicon

## Asset Guidelines

- **Icon**: Should be a square image (1024x1024px) with no transparency for iOS
- **Splash Screen**: Should cover the entire screen, typically 1284x2778px or larger
- **Adaptive Icon**: Android adaptive icon (foreground image)
- **Favicon**: Small icon for web version

You can generate these assets using tools like:
- [Expo Asset Generator](https://www.npmjs.com/package/@expo/config-plugins)
- Online tools or design software

For now, placeholder files can be created, or you can use Expo's default assets during development.
